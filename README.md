# Billed-app

Application SaaS de gestion des notes de frais pour les ressources humaines.

## Description

Application full-stack permettant aux employés de soumettre leurs notes de frais et aux administrateurs de les valider. Développée avec une architecture frontend/backend séparée.

## Démarrage

### 1. Backend

```bash
cd backend
npm install
npm run run:dev
```

Le serveur démarre sur `http://localhost:5678`

### 2. Frontend

Ouvrez [`index.html`](index.html) dans votre navigateur ou utilisez Live Server dans VSCode.
